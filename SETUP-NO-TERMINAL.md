## 🚀 **3-Minute Setup (No Terminal!)**

### Step 1: Create GitHub Repo
1. Go to **github.com** → Click **New Repository**
2. Name it: `flexi-dev-studio`
3. Make it **Public**
4. Click **Create Repository**

### Step 2: Upload Files
1. Click **Add file** → **Upload files**
2. Upload these 3 files:
   - `index.html`
   - `sw.js`
   - `manifest.json`
   - `vercel.json` (optional, for Vercel)
3. Click **Commit changes**

### Step 3: Deploy to Vercel
1. Go to **vercel.com** → Sign up with GitHub
2. Click **New Project**
3. Select your `flexi-dev-studio` repo
4. Click **Deploy**
5. **Done!** 🎉 Your app is live!

## 🔄 Making Changes

### To Update Content:
1. Edit `index.html` in GitHub
2. Scroll down to find the data you want to change
3. Edit (search for projects, tasks, etc)
4. Commit changes
5. **Vercel auto-deploys** ✨

Example: Change project names:
```javascript
const projects = [
  { icon: '📊', name: 'YOUR PROJECT NAME', ... }
];
```

### To Change Colors:
Search for `:root` in CSS, change:
```css
:root {
  --dark: #0a0e27;      /* Main dark color */
  --blue: #0066ff;      /* Main blue */
  --blue-light: #0099ff; /* Light blue */
}
```

## 🚨 Troubleshooting

### App doesn't show after deploy?
- Wait 2-3 minutes for Vercel
- Hard refresh (Ctrl+Shift+R)
- Clear browser cache

### Chat not working?
- It's demo data (pre-loaded)
- Type → sends locally
- Refreshing page resets messages (expected)

### Install button not showing?
- Need HTTPS (Vercel has this ✓)
- Desktop browsers may hide button
- Works great on mobile!

### PWA not installing?
- Mobile browser required
- HTTPS required (Vercel ✓)
- Use Chrome or Safari
- Try adding manually: Menu → Install App
